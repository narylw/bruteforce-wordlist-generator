# bruteforce-wordlist-generator
Golang Word List Generator For Brute Force

For Run It In Kali/Termux:
> sudo su

> apt update

> apt upgrade

> apt install git

> apt install go

/or/

> apt install go-lang

> git clone https://github.com/narylw/bruteforce-wordlist-generator

> cd bruteforce-wordlist-generator

> go run brute.go
